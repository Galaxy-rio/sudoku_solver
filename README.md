# README

A step by step sudoku solver.

## Techniques

### Supported techniques:
- Hidden Single
- Naked Single
- Locked Candidate
- Hidden Pair
- Hidden Triple
- Hidden Quad
- Naked Pair
- Naked Triple
- Naked Quad
- X-Wing
- Swordfish
- Jellyfish
- Color Trap
- Color Wrap
- Color Wing

- XY-Wing
- XYZ-Wing
- XY-Chain
- Finned X-Wing
- Finned Swordfish
- Finned Jellyfish
- Empty Rectangle
- Aligned Pair Exclusion
- Almost Locked Set
- Unique Rectangle
